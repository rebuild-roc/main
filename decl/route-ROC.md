海內外關心中國前途的各位朋友：

專制政權的垮臺經常是以意外方式發生的，請容在下從晚唐章碣那首名詩《焚書坑》說起，
全詩如下：
```
   竹帛煙消帝業虛，關河空鎖祖龍居。
   灰坑未冷山東亂，劉項原來不讀書。
```
這首詩是說，秦始皇本來以爲，
燒了諸子百家的書就可以保證人心不亂，秦朝就可以萬世一統了，
沒想到秦帝國二世而亡，而且推翻秦帝國的劉邦和項羽根本就不是讀書人。

秦始皇萬般防範諸子百家和列國貴胄，又盡收天下兵器，
沒想到陳勝吳廣率領一班赤手空拳的戍卒揭竿而起，曾經席捲天下的秦帝國就此轟然倒地，
真可謂是出其不意！

事實上，又豈止秦朝如此？蘇聯垮臺的時候不也是一樣出乎所有人的意料？
而今中共也開始風雨飄搖了，我想說，中共也會以超乎常規的方式垮臺，
而不是人們一直期待的東歐模式和前蘇聯模式，當然更不可能是秦朝那種模式。
這樣推測的理由很簡單：什麼民變、兵變、政變，中共是這方面的行家裏手，
它竊國之後的種種作爲，說到底就是要防止自己因爲民變、兵變和政變而垮臺；
有衆多要推翻中共的志士偏偏要從這些地方着手，正是「以己之短，攻敵之長」，
雖不敢說其敗必然，但是說其勝算渺茫不爲偏頗。

既然中共不太可能垮於民變、兵變和政變，那麼它該垮於什麼因素？
實際上，中共的垮臺方式已經漸露端倪：
先有一大批年輕人不願意繼續當韭菜，採取各種方式躺平；
繼而有上海一些企業家和投資人發公開信要中共改弦更張，
否則也會從暫時躺平改爲永久躺平；
最近全國又有前赴後繼的大批業主要斷供掉價的物業，或者停貸爛尾的期房
——而這個現象的背後，是那些開發商們的先期逃離。
經過習近平十年的加速倒車，如今財政喫緊已經不是中共當局的主要危險，
更危險的是整個政權要隨着金融系統的崩潰而分崩離析。
習近平奉持前清醇親王載灃的「有兵在」這句亡國之音，
兀自揮舞鐮刀肆意收割，卻不去想中共的鐮刀是一口嗜血的邪門兵器，
割不到韭菜就要反噬揮刀人。

就是說，**推翻中共的模式不一定是民衆上街、各界罷工罷市罷學等等，
更不是民衆拿起刀槍把中共推翻，而是在經濟領域發動一場無聲無息的戰爭，
用一種新的貨幣替代中共的「人民幣」，最後導致中共失去經濟控制權、財政枯竭，
被其龐大的官僚、公務員隊伍和軍隊、警察拖垮，甚至被其軍隊和警察反噬。**
這是一場新型的戰爭，戰爭主要發生在網絡空間中，戰爭爭奪的焦點是全國的最高信用、
經濟控制權、貨幣的角逐等等。
在這場戰爭中，中共龐大的軍隊不但沒有用處，反而是負累。
故，不才拼湊一首打油詩《兵何及》奚落中共，忝與《焚書坑》湊趣：
```
    毛病不除成惡習，刀槍嚴陣民積疲。
    硝煙未起江山易，四海無獲兵何及？
```

我們不能坐等經濟危機把中共摧垮，而是要藉經濟危機之良機推行新貨幣，
利用中共的倒行逆施助推「人民幣」的貶值，使中國民衆逐漸拋棄「人民幣」。
民衆拋棄了「人民幣」，中共就失去了經濟控制權；中共失去了經濟控制權，
財政就會枯竭，做惡就沒有了資本，政權就會迅速垮臺。
在這個過程中，廣大民衆都能安居樂業，
新的國家政權也可以在中共垮臺前開始逐漸孕育和成長，等到時機成熟，
只要把中共軍隊收編爲新政權的國家軍隊，
新的國家政權就可以迅速由影子政府轉變爲實體政府，
於是，在整個社會轉型期間也就不至於發生劇烈的動盪。

上述只是一個總體戰略。
自比特幣出現以來，本人就開始構思這樣一個戰略，以期不但能摧毀中共，
而且還要避免中共垮臺後出現軍閥割據和軍閥混戰，並要保證國家不再重蹈專制泥淖。
今將構思略陳如下，以供各路英才批評、改進，兼答[上海部分企業家和投資人的公開信](Shanghai-20220530.md)。


## 一、民變、兵變、政變論可以休矣

我不贊同以民變-兵變-政變方式推翻中共，主要的原因不是它的難度，
而是其結果的不確定性：
1. 這種方法能不能推翻中共？
2. 如果中共倒臺了，能否保證中國社會不發生動盪，尤其是不會發生軍閥割據和軍閥混戰？
3. 中共倒臺之後，接下來的政權就一定是民主政權？能否保證接手的政權不是專制政權？

這麼多年來，沒有見到哪位持民變-兵變-政變論者給出什麼可行的方案能夠一體化地解決上述三個問題。

我承認，直到今天爲止，從理論到實踐，社會轉型的主要手段仍然是民變、兵變和政變。
如果沒有更好的策略，那麼，即使存在上述問題，中國的社會轉型還是要被迫走民變、兵變和政變的老路。
但是，現在，我想從理論上先構想一下，我們應該有更好的辦法，這就是我前面所說的戰略
——這個戰略的主軸就是與中共爭奪中國大陸的經濟控制權，從財政上逐步困死中共。

這樣的戰略構想在 2009 年之前是不可行的。
但是，2009 年 1 月 3 日比特幣正式誕生和運行，這個事件意味着，
**人類爭奪政權的方式可以從物理世界中的刀兵相向轉變爲網絡世界中的貨幣對決。**
我想從以下幾個方面展開討論這一點。

* 其一，爭奪政權的根本是爭奪相關領土上的經濟控制權。

   大家只要藉助於常識就可以理解，
   一個政權的生命力來自它對它所宣稱的領土上的經濟控制權，
   而不是毛澤東所謂的「槍桿子裏面出政權」。
   例如，土匪手裏有槍，可以靠打家劫舍維生，
   但是，只要土匪一日不能控制它周圍區域的經濟，則土匪一日不能建立政權，
   其生存成本和風險就會居高不下；
   相反，如果能夠控制一個區域中的經濟，哪怕是像黑社會一樣只是部分控制，
   那就可以依靠收稅或收「保護費」而維持穩定的收入，不用整天刀頭舔血。

   中共政權是從井岡山匪巢發展起來的，至今還帶有半土匪半政權的特性，
   就與毛澤東的這個見識有關，民主運動人士不可不察，
   以致因畏懼中共的武力而裹足不前，抑或爲尋求武力而誤入歧途。

   從鄧小平主政到江、胡時期，中共的匪氣略少了那麼一點；
   毛澤東和習近平主政時期則是匪氣沖天，習近平獵殺民營經濟、動輒毀壞一個行業，
   就是井岡山紅匪的升級版。
   他的這種作爲是很使社會中產階級和富豪階層從「改革開放」的迷夢中驚醒，
   使他們深刻認識到，中共永遠都不事生產，不會容許歲月靜好，
   時候一到就會劫掠社會，其土匪本色是永遠不可改變的。
   這爲推翻中共政權打下了堅實的社會基礎。

   要奪取經濟控制權，只要社會的中產階級和富豪階層起而配合，其勢將如脫繮野馬、
   決堤洪水，必能將中共政權摧枯拉朽一般沖垮。

* 其二，因爲經濟運行的基本工具就是貨幣，所以，要爭奪經濟控制權，
   其中一個選項就是以貨幣戰爭爲主軸。

   對現政權或準政權發動貨幣戰爭，這在信息時代之前是不可想象的，因爲，在那時，
   貨幣以武力爲後盾維持其信用。
   到了信息時代，由於加密貨幣及其背後的各種信息技術，
   以及電子商務這種商業模式和它後面的網絡支撐，發動貨幣戰可以在網絡空間中展開，
   這就是現政權或準政權的武力大打折扣。
   更甚者，現在還有各種各樣的暗網，這就可以進一步降低中共「刀把子」的效用。

   以比特幣爲肇始和代表的加密貨幣還深刻地揭示了貨幣的本質：
   貨幣不是通用交換物，也不是通用交換物的代表或符號，
   因爲加密貨幣的後面就沒有「物」；
   **貨幣是一種能夠被社會認可的量化的信用，
   人們憑這種量化的信用實現資源（包括商品、信息、知識技能等）以及服務的交換。**

   加密貨幣能夠脫離政權而通行，
   充分證明貨幣可以不依靠武力、僅依靠信用就可以成其爲貨幣。
   所以，到了信息時代，貨幣戰爭的本質就是信用爭奪戰。
   與中共比拼武力，就連美國都要躊躇再三；與中共比拼信用，則中共必敗無疑。

* 其三，因爲貨幣只是龐大的信用系統中一個重要組成部分而已，所以，
   在與中共「人民幣」的貨幣戰爭也不是孤立的戰爭，必須有龐大的支撐系統。

   在中共這一方，中共有各種武力，還有一幫附庸的筆桿子和小粉紅、老粉紅，
   他們構成貨幣戰爭的一個支撐系統；
   對應地，我們必須有依託於信息技術的各種系統，以便壓制中共的貨幣戰支撐系統。
   舉例來說，我們需要有推行新貨幣的網上和網下機構；
   我們需要有足夠的震懾手段去打擊假冒僞劣產品的生產、銷售；
   我們需要有足夠的手段保證交易的公平和公正；
   我們需要有足夠的手段震懾幫助中共釣魚的生產商、運輸商、銷售商和消費者，
   以保護使用新貨幣的生產商、運輸商、銷售商和消費者；
   我們還要有足夠的機制用於監督市場上流通商品的質量；
   等等。總而言之，單純地推出一個新貨幣不足以推翻中共，必須要有一個合成的系統。

現有的信息技術和人工智能已經可以部分地實現影子政府的職能。
這些職能有些可以完全實現非人工化，例如交易的結算和相關的信用記錄；
有些可能需要人工操作，但是有了去中心化技術和網絡匿蹤技術，
完全不用擔心操作人員遭到中共追捕。
這些職能系統，或者徑直叫做影子政府，將會成爲貨幣戰中的一個重要的支撐系統。
待到影子政府的功能足以維持社會經濟的良好運轉、
同時中共的「人民幣」也已經落敗的條件下，
就可以考慮收編中共的軍隊，然後就把影子政府的職能逐步、逐個地區地移交給新政權的臨時政府（其中可以是中共政權中歸順過來的看守政府）。
待到臨時政府的領地足夠大、收編的軍隊足夠多足夠強之後，就可以宣告新政權的建立，
並開始準備正式的國會選舉和總統選舉。
相關的具體流程後面還會詳細討論。
這樣按部就班地進行，可以保證整個國家在社會轉型期間不至於發生劇烈的社會動盪，
保證一切都按照既定的方向發展。

新貨幣的流通和結算系統的最終決定權將永遠保留在公民手中。
簡要來說，全體公民的多數能夠決定新貨幣的控制權在誰手上。
這樣，一旦出現國民不滿意的政府，就可以通過剝奪貨幣控制權的方式逼迫該政府下臺。
這是維護民主體制的有效手段，
比美國的公民持槍權更能夠有效地壓制高科技和大資本對社會的控制，
傳統的獨裁政客要想竊國將會難如登天。

顯然，如果按照這樣的構想來實現中國的社會轉型，那麼不但轉型的結果是可預見的，
而且轉型的過程也將會發生很小的生命代價。
如中共那樣以兩千三百萬顆人頭（如果算上無辜老百姓的生命，這個數字還會更大）的代價
鑄就一個政權，而且這個政權還是個邪惡專制的政權，這樣的悲劇將不會發生。
正是在這個意義上，我要說，民變-兵變-政變論可以休矣。
